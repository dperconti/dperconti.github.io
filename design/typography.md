# Typography System

## Font Families

### Primary Font: Inter
- **Type**: Sans-serif
- **Usage**: Headings, hero text, large display elements
- **Weights**: 400 (Regular), 500 (Medium), 600 (Semi-bold), 700 (Bold), 800 (Extra-bold), 900 (Black)
- **Characteristics**: Clean, modern, professional, highly readable
- **Best For**: Main headings (H1), hero sections, page titles, all UI elements

### Secondary Font: Inter
- **Type**: Sans-serif
- **Usage**: Body text, paragraphs, UI elements
- **Weights**: 400 (Regular), 500 (Medium), 600 (Semi-bold), 700 (Bold)
- **Characteristics**: Clean, modern, professional, versatile
- **Best For**: Body copy, descriptions, navigation, buttons, all text elements

## Type Scale

Based on a base font size of 16px with a scale ratio of 1.250:

- **H1**: 3.052rem (48.83px) - Hero titles, main page headings
- **H1 Small**: 2.441rem (39.06px) - Responsive hero titles
- **H2**: 2.441rem (39.06px) - Section headings
- **H2 Small**: 1.953rem (31.25px) - Responsive section headings
- **H3**: 1.953rem (31.25px) - Subsection headings
- **H3 Small**: 1.563rem (25px) - Responsive subsection headings
- **H4**: 1.563rem (25px) - Card titles, small headings
- **H5**: 1.25rem (20px) - Small headings, labels
- **H6**: 1rem (16px) - Smallest headings, captions
- **Base**: 1rem (16px) - Body text, default size

## Font Weights

- **Regular (400)**: Body text, descriptions
- **Medium (500)**: Labels, emphasis
- **Semi-bold (600)**: Buttons, CTAs, important text
- **Bold (700)**: Strong emphasis, highlights

## Line Height

- **Headings**: 1.2 - Tight, impactful
- **Body Text**: 1.6-1.8 - Comfortable reading
- **Large Text**: 1.4 - Balanced for larger sizes

## Letter Spacing

- **Headings**: 0.02em - Slight spacing for clarity
- **Body**: 0 - Natural spacing
- **Uppercase**: 0.05em - For labels and buttons

## Typography Hierarchy

### Hero Section
- **Title**: H1, Abril Fatface, Bold
- **Subtitle**: H3 or large body, Bitter, Medium
- **Description**: Body, Bitter, Regular

### Content Sections
- **Section Title**: H2, Abril Fatface, Regular
- **Subsection**: H3, Bitter, Semi-bold
- **Body**: Base, Bitter, Regular
- **Emphasis**: Base, Bitter, Medium or Semi-bold

### Navigation
- **Menu Items**: Base, Bitter, Medium
- **Active State**: Base, Bitter, Semi-bold

### Buttons & CTAs
- **Button Text**: Base or H6, Bitter, Semi-bold
- **Large CTA**: H5, Bitter, Bold

## Responsive Typography

- **Mobile**: Smaller scale (0.8x multiplier)
- **Tablet**: Standard scale
- **Desktop**: Full scale with generous spacing

## Usage Guidelines

1. **Limit Font Mixing**: Use primary font for headings only, secondary for everything else
2. **Maintain Hierarchy**: Clear distinction between heading levels
3. **Readability First**: Ensure text is readable at all sizes
4. **Consistent Spacing**: Use consistent margins between headings and content
5. **Theme Adaptation**: Text colors adapt to dark/light themes automatically

## Glass Morphism Typography

When text appears on glass cards:
- **Headings**: Maintain high contrast, may use subtle text shadow in dark theme
- **Body**: Slightly increased opacity for readability
- **Links**: Use accent colors (red/green) sparingly for emphasis
